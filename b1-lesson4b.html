בית›B1›שיעור 4 — Instrumentál›4ב — Instrumentál יחיד
🛠️ Instrumentál יחיד — הסיומות בשורה הטובה: Instrumentál יחיד פשוט יחסית. זכרים ונטרלים כמעט תמיד מקבלים -em. נקבות -a מקבלות -ou. נקבות -e/-ce ועיצורים מקבלות -í.
סוג המילהסיומתNom יחיד → Instr יחידדוגמה
🔵 זכר (Masculine) — כמעט תמיד -em
זכר קשה (חי ודומם)
-em
Petr → Petrem
kamarád → kamarádem
park → parkem
hotel → hotelem
 s Petrem · parkem
זכר רך (-č/-š/-ž/-ř/-c)
-em
muž → mužem
lékař → lékařem
počítač → počítačem
klíč → klíčem
 s mužem · klíčem
🔴 נקבה (Feminine)
נקבה -a (קשה ורכה)
-ou
žena → ženou
škola → školou
Praha → Prahou
banka → bankou
kavárna → kavárnou
 s ženou · školou
נקבה -e / -ce (רכה)
-í
restaurace → restaurací
ulice → ulicí
práce → prací
tramvaj → tramvají*
 tramvají · s prací
נקבה עיצור בסוף
-í
věc → věcí
radost → radostí
noc → nocí
loď → lodí
 lodí · s věcí
⚪ נטרל (Neuter) — כמעט תמיד -em
נטרל -o קשה
-em
město → městem
auto → autem
okno → oknem
víno → vínem
 autem · s vínem
נטרל -um
-em
muzeum → muzeem
centrum → centrem
metro → metrem
 metrem · muzeem
נטרל -í (רך)
-ím
náměstí → náměstím
nádraží → nádražím
 před nádražím
⚠️ tramvaj — שימו לב! tramvaj היא נקבה עם j בסוף (כמו נקבה רכה). Instrumentál: tramvají (לא tramvajou!)
זה החריג הנפוץ ביותר — כי משתמשים בו כל יום בפראג.

דוגמאות לשמיעה

Jedu autobusem. נוסע באוטובוס (autobus → autobusem)
Jdu s kamarádem. הולך עם חבר (kamarád → kamarádem)
Přijel vlakem. הוא הגיע ברכבת (vlak → vlakem)
Jedu tramvají. נוסע בטרם (tramvaj → tramvají)
Jsem učitelkou. אני מורה (אישה) (učitelka → učitelkou)
Letí letadlem do Londýna. הוא/היא טס/ה במטוס ללונדון
🎯 הכלל הפשוט לזכור זכר + נטרל → -em (כמעט תמיד)
נקבה -a → -ou (ženou, školou, Prahou)
נקבה -e/-ce + עיצור → -í (tramvají, lodí, věcí)
→ 4א — שימושים 4ג — פעלים עם Instrumentál ←
📋 Instrumentál יחיד — סיכום

-í — נקבה -e/-ce + עיצור
