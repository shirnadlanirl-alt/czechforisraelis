5ב: Dativ יחיד — סיומות

סיומות לפי מין: זכר animate → -ovi/-u, נקבה -a → -ě, נטרל → -u.

📋 Dativ יחיד — סיומות לפי מין כל מין ותת-קטגוריה מקבלים סיומת ייחודית. הטבלה הזו היא הלב של שיעור 5. הכלל לזכירה: זכר animate → -ovi/-u · נקבה -a → -ě (עם מוטציות!) · נטרל → -u.

🟦 זכר — Mužský rod

קטגוריהשורש → Dativסיומתעוד דוגמאות
זכר animate (אנשים/בעלי חיים)
hard animatekamarád → kamarádovi-ovi / -uPetrovi, otci, bratrovi, tatínkovi
זכר inanimate (דברים)
hard inanimatepark → parku-uhotelu, týdnu, měsíci
זכר soft (muž, lékař)
softmuž → muži-ilékaři, průvodci, řidiči
⚠️ -ovi לעומת -u לזכר animate שתי הצורות נכונות: Petrovi = Petru. בכתיבה רשמית -ovi מועדף. בדיבור -u שכיח. כשיש שמות ברצף — -u לכולם חוץ מהאחרון: Janu Petru Novákovi.

🔴 נקבה — Ženský rod

קטגוריהשורש → Dativסיומתעוד דוגמאות
נקבה ב-a (žena, škola, Praha)
hard -ažena → ženě-ě / -eškole, mamince, Praze, bance
נקבה ב-ce/-ze/-je (restaurace, ulice)
-ce / -zerestaurace → restauraci-iulici, práci, tramvaji
נקבה על עיצור (věc, radost, noc)
עיצורvěc → věci-iradosti, noci, nemoci, lodě

⚡ מוטציות עיצורים לפני -ě (נקבה hard ב-a)

לפני הסיומת -ě חלק מהעיצורים האחרונים של הגזרה משתנים — בדיוק כמו ב-Lokál!

k→c: banka → bance, Praha → Praze... רגע, h→z
h→z: Praha → Praze, kniha → knize
ch→š: střecha → střeše
r→ř: sestra → sestře, dcera → dceři
k→c: banka → bance, kamarádka → kamarádce, mama → mamince (חריג!)

🟢 ניטרלי — Střední rod

קטגוריהשורש → Dativסיומתעוד דוגמאות
ניטרלי ב-o (město, auto, okno)
-oměsto → městu-uautu, oknu, slovu, vínu
ניטרלי ב-um (muzeum, centrum)
-ummuzeum → muzeu-ucentru, metru, fóru
ניטרלי ב-í (náměstí, nádraží)
-ínáměstí → náměstí-í (ללא שינוי)nádraží, učebnici... ⚠️

דוגמאות לשמיעה

Dám to Petrovi.אתן את זה לפטר
Řeknu to mamince.אספר לאמא
Jdu k restauraci.אני הולך למסעדה
Pomohl jsem kamarádce.עזרתי לחברה
Jsem blízko městu.אני קרוב לעיר
🎯 קיצור זיכרון לניטרלי ניטרלי ב-o וב-um תמיד מקבלים -u ב-Dativ. ניטרלי ב-í — ללא שינוי.
💡 k לעומת ke ke לפני צרורות עיצורים: ke škole, ke kamarádovi, ke zdi.
k לפני עיצור בודד: k lékaři, k moři, k tobě.

📋 סיכום סיומות יחיד
Masc animate: -ovi / -u
Masc inanimate: -u
Masc soft: -i
Fem -a: -ě / -e
Fem -ce/-ze: -i
Fem עיצור: -i
Neut -o/-um: -u
Neut -í: -í (ללא שינוי)
