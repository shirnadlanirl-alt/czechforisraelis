שיעור 4: Instrumentál — יחסה 7

יחסת האמצעי, הלוואי והתפקיד. Jedu autobusem · Jdu s kamarádem · Jsem učitelkou · Píšu tužkou.

🛠️
Instrumentál — יחסת האמצעי Instrumentál מבטאת במה (Jedu autobusem = נוסע באוטובוס), עם מי (Jdu s kamarádem = הולך עם חבר), ומה תפקידי (Jsem učitelkou = אני מורה). יחסה אחת, שלושה שימושים.
🛠️ מה Instrumentál עושה? השאלות שמזהות אותה: ״עם מי/מה? / במה? / כמה?״ (s kým? čím?)
שלושה שימושים עיקריים: s/se (עם), אמצעי תחבורה, ומקצועות עם být. בנוסף: אחרי פעלים כמו stát se, stávat se, zabývat se, zajímat se.

שימושים עיקריים

s / se עם (לוואי)
autobusem באוטובוס
tramvají בטרם
být + Instr מקצוע/תפקיד

מה Instrumentál עושה?
שלושת השימושים: s/se + לוואי, אמצעי תחבורה (autobusem, tramvají, autem), ומקצועות עם být. הסבר עם דוגמאות.

4ב
Instrumentál יחיד — סיומות
זכר ונטרל → -em, נקבה -a → -ou, נקבה -e/-ce → -í, נקבה עיצור → -í. טבלה מלאה עם דוגמאות לכל סוג.
⏱ 13 דקות טבלה, כל המינים

פעלים עם Instrumentál
הפעלים שדורשים Instrumentál: stát se · zabývat se · zajímat se o · stávat se · být. עם דגש על ההבדל בין stát se ל-stávat se.

Instrumentál רבים
זכר → -y/-i, נקבה → -ami/-emi, נטרל → -y. טבלה עם כל הסוגים ודוגמאות.
