בית›B1›שיעור 4 — Instrumentál›4ד — Instrumentál רבים
🛠️ Instrumentál רבים — הסיומות ב-Instrumentál רבים יש שתי סיומות עיקריות: -y/-i לזכרים ונטרלים, ו-ami/-emi/-mi לנקבות. הצורות האלה פחות נפוצות בדיבור יומיומי — אבל חשובות לכתיבה ולהבנה.
סוג המילהInstr רביםNom יחיד → Instr רביםדוגמה
🔵 זכר קשה — -y
זכר קשה (חי ודומם)
-y
kamarád → kamarády
student → studenty
park → parky
hotel → hotely
 s kamarády · parky
🔵 זכר רך — -i
זכר רך (-č/-š/-ž/-ř)
-i
muž → muži
lékař → lékaři
počítač → počítači
klíč → klíči
 s muži · s lékaři
🔴 נקבה -a — -ami
נקבה -a (קשה ורכה)
-ami
žena → ženami
škola → školami
kavárna → kavárnami
banka → bankami
 se ženami · školami
🔴 נקבה -e/-ce — -emi
נקבה -e / -ce (רכה)
-emi
restaurace → restauracemi
ulice → ulicemi
tramvaj → tramvajemi*
 s restauracemi · tramvajemi
🔴 נקבה עיצור — -mi
נקבה עיצור בסוף
-mi
věc → věcmi
radost → radostmi
loď → loďmi
 s věcmi
⚪ נטרל -o — -y, נטרל -í — -ími
נטרל -o קשה
-y
město → městy
auto → auty
okno → okny
 s auty · městy
נטרל -í
-ími
náměstí → náměstími
nádraží → nádražími
 před náměstími
💡 בשפה מדוברת: -ama / -ema בצ'כית מדוברת (hovorová čeština) נשמע לעתים קרובות:
s kamarádama (במקום s kamarády) · se ženama (במקום se ženami)
זה נפוץ בשיחה יומיומית אבל לא סטנדרטי בכתיבה.

דוגמאות לשמיעה

Šel jsem s kamarády do kina. הלכתי עם חברים לקולנוע
Mluvil se ženami. הוא דיבר עם נשים
Přijeli jsme auty. הגענו באוטואים (auty = רבים של autem)
