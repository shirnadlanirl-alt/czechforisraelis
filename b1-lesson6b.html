6ב: Nominativ רבים — סיומות ושינויים

טבלה מלאה: גזע קשה → -i (עם שינוי), גזע רך → -é, צורות לא סדירות.

📋 סיומות Nominativ רבים — זכר חי (Ma) שתי קבוצות עיקריות: (1) גזע קשה → -i עם שינוי עיצור אפשרי, (2) גזע רך → -é. בנוסף יש כמה צורות לא סדירות שצריך לזכור בנפרד.
⚠️ שינויי גזע לפני -i (גזע קשה בלבד) כשמוסיפים -i לגזע קשה, העיצור האחרון יכול להשתנות:
k → c
h → z
ch → š
r → ř
לא כל הגזעים הקשים עוברים שינוי — student → studenti (t נשאר). רק k, h, ch, r משתנים.

גזע קשה — ← -i (ללא שינוי גזע)

-d (קשה) prezident -i presidenti נשיאים

גזע קשה — שינוי עיצור + -i
-k → c Slovák -ci (-k→c) Slovák → Slováci סלובקים
-ch → š Čech -ši (ch→š) Čech → Češi צ'כים
-r → ř doktor -ři (r→ř) doktor → doktoři רופאים
-r → ř bratr -ři (r→ř) bratr → bratři אחים
גזע רך — -é
-ž (רך) muž -i / -é* muži גברים
-l (רך) učitel -é učitelé מורים
-ec Němec -ci Němec → Němci גרמנים
לא סדיר — חובה לזכור
לא סדיר přítel -é (שינוי גזע) přítel → přátelé חברים
לא סדיר člověk שינוי גזע člověk → lidé אנשים
-an občan -é občané אזרחים
-pán pán -i / -é pán → páni / pánové אדונים
דוגמאות לשמיעה
Slováci a Češi jsou přátelé.
סלובקים וצ'כים הם חברים. (k→c, ch→š)
Doktoři pracují v nemocnici.
הרופאים עובדים בבית החולים. (r→ř)
Tito učitelé jsou velmi dobří.
המורים האלה מאוד טובים. (גזע רך -l → -é)
Lidé jsou různí.
אנשים הם שונים. (člověk → lidé — לא סדיר)
✅ הצורות הכי חשובות לזכור studenti (סטודנטים), Češi (צ'כים), doktoři (רופאים), muži (גברים), učitelé (מורים), přátelé (חברים), lidé (אנשים). אלה מכסים את רוב המצבים שתיתקל בהם.
⚠️ תשומת לב ל-pán ו-přítel pán (אדון) → páni (בשיחה רשמית) או pánové (פנייה לקהל). přítel (חבר) → přátelé (שינוי תנועה בגזע + -é). שתי הצורות שכיחות מאוד בחיי היום-יום.
