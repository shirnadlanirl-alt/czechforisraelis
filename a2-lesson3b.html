🎯 מה נלמד בדף הזה? פעלים שצועדים תמיד עם המילה הקטנה se — כמו dívat se (לראות) או učit se (ללמוד). בעבר זה יוצר שאלה: איפה ה-se הולך? התשובה מפתיעה בפשטות שלה.
מה זה פועל רפלקסיבי?
פעלים שהולכים יד-ביד עם se
בעברית יש לנו פעלים שמתחילים בבניין "התפעל": התלבשתי, התרגזתי, התקלחתי. הם תמיד מתייחסים לפועל שעושים על עצמך.

בצ׳כית התפקיד הזה ממלאת מילה נפרדת — se. היא חלק קבוע מהפועל, ואי-אפשר להפריד ביניהם: dívat se, učit se, bavit se.

בעבר זה יוצר שאלה: יש לנו שני חלקים שצריכים להיות במקום השני — jsem וגם se. איך זה עובד?
הכלל: jsem se מתאחדים
💡 כשיש גם jsem וגם se הסדר תמיד jsem se (לא se jsem!). שניהם נצמדים אחרי המילה הראשונה במשפט:
Včera jsem se díval na film. — אתמול ראיתי סרט.

ארבעה פעלים רפלקסיביים שימושיים
dívat se (na) ← díval / dívala se להסתכל על / לצפות ב-
Včera jsem se dívala na seriál. אתמול צפיתי בסדרה. ♀
Na co jsi se díval o víkendu? במה צפית בסוף השבוע? ♂
učit se ← učil / učila se ללמוד (לעצמך)
Učila jsem se česky tři roky. למדתי צ׳כית שלוש שנים. ♀
Včera večer jsme se učili na test. אתמול בערב למדנו למבחן.
bavit se ← bavil / bavila se להעביר זמן טוב / לפטפט
Bavil jsem se na party celý večer. העברתי זמן טוב במסיבה כל הערב. ♂
Bavily jsme se s kamarádkami. פטפטנו עם החברות. ♀ (קבוצת נשים)
vrátit se ← vrátil / vrátila se לחזור (ממקום)
Včera večer jsem se vrátila pozdě. אתמול בערב חזרתי מאוחר. ♀
Kdy jste se vrátili z dovolené? מתי חזרתם מהחופשה?
