9א: מה Imperativ?

שלוש צורות הציווי: יחיד, רבים/כבוד, והצעה משותפת. הנוסחה מגוף 3 רבים.

📢 Imperativ — ציווי בצ'כית ציווי = לצוות, לבקש, להציע. בצ'כית יש שלוש צורות: יחיד (לאדם אחד), רבים / כבוד (לכמה או בצורה מנומסת), ו-"בוא נ..." (הצעה לפעולה משותפת). הבסיס — גוף שלישי רבים בהווה.
שלוש צורות הציווי
צורהיצירהדוגמהמשמעות
יחיד (ty) בסיס Imperativ Čti! קרא! (לאדם אחד)
רבים (vy) יחיד + -te Čtěte! קראו! (לכמה / כבוד)
"בוא נ-" (my) יחיד + -me Čtěme! בואו נקרא!
איך מוצאים את הבסיס? — מגוף 3 רבים
הנוסחה: oni (הם) בהווה → הסר -í/-ou/-ají → זה הבסיס číst → oni čtou → čt → Čti!
dělat → oni dělají → dělaj → Dělej!
mluvit → oni mluví → mluv → Mluv!
דוגמאות לשמיעה
Mluv pomalu!
דבר לאט! (יחיד)
Mluvte prosím pomalu.
דברו לאט בבקשה. (רבים/כבוד)
Pojďme domů.
בואו נלך הביתה. (הצעה משותפת)
Dej mi to!
תן לי את זה! (יחיד)
Poslouchejte pozorně.
הקשיבו בתשומת לב. (רבים/כבוד)
💡 prosím = "בבקשה" מרכך הכל הוספת prosím הופכת כל ציווי לבקשה מנומסת: Dej mi to, prosím. / Počkejte prosím. בשפה היומיומית — prosím חיוני!
